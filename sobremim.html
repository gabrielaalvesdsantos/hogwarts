Sobre Mim

Olá! Meu nome é Gabriela (ou Gabi), tenho 15 anos e este projeto foi desenvolvido para a disciplina de Laboratório de Programação para Web do curso de Informática (CEFET-MG).

Adoro ler (ficção, distopia e romance), assistir filmes (MCU, HP, DCU e qualquer outro tipo de filme), assistir F1 e acompanhar futebol.

Estou sempre buscando melhorar minhas habilidades.

Para a execução desse site tive como referência os sites 'Fandom' e 'Legião dos Heróis' para o conteúdo do site e usei como meio de ajuda para programar os antigos slides do meu professor e também o 'Developer Mozilla'.

Usei recursos de terceiros para as minhas imagens (Pinterest) e meu vídeo (Youtube, canal Harry Potter).

Na execução do programa utilizei HTML, CSS e JavaScript, aplicando conceitos de interatividade, responsividade e design moderno.

Tecnologias Utilizadas
